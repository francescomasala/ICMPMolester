# Security Policy

## Supported Versions

Only the latest release receives security updates. Please update to the newest
tag before requesting assistance.

## Reporting a Vulnerability

If you discover a vulnerability or suspect a security issue, please email
`mail@francescomasala.me` with:

- A detailed description of the issue.
- Steps to reproduce or proof-of-concept code where possible.
- Your contact information for follow-up.

I aim to acknowledge reports within five business days.
